##1.命令列表#############################################################
#/bekit deletekit --删除你在当前世界的套件文件
#/bekit loadkit --加载你在当前世界的套件
#/bekit loaddefaultkit --加载对于当前世界默认的套件
#/bekit savekit --将你当前物品栏保存为你在当前世界的套件
#/bekit about --查看插件相关信息
#/bekit help --查看插件指令帮助
##2.默认套装说明书########################################################
#插件加载时会生成名为"default_example.yml"的文件
#这是一个示例套件文件，它不能被插件作为默认套件加载
#其文件名该如此解读：

#"default"代表它是一个默认套件文件
#"example"代表它是一个示例文件

#插件加载默认套件的运作方式为：
#将"example"的位置作为一个世界的名称读取
#如果你想要为一个世界"pvp"创建默认套件，你应该将复制文件并命名为

#default_pvp.yml

#并将其存到该插件的文件根目录下

#plugin/BeKit/default_pvp.yml
